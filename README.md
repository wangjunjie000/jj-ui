# jj-vue-ui

仿 element 开源组件库。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
生成 组件库 文档
```
npm run build
```

### 生成 npm 库文件
记得生成前将 package.json 中的 version 版本号加 1
```
npm run lib
```

### 发布到 npm 库
```
npm run publish
```
